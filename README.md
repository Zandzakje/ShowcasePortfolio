# ShowcasePortfolio
This project is used to show my various work and contact information for internship purposes.\
Visit the current live version here: https://zandzakje.github.io/ShowcasePortfolio

## Features
- Home section that acts as an introduction to the showcase portfolio
- About section with highlighted works
- Contact section with various ways to contact me

## Getting started
### Requirements
- Node.js (v24 or higher)
- npm

### Installation
1. Clone the repository
```
git clone https://github.com/Zandzakje/ShowcasePortfolio.git
cd showcaseportfolio
```
2. Install dependencies
```
npm install
```
3. Start the application
```
npm start
```
4. Copy paste into your browser to see the app
```
http://localhost:3000
```
